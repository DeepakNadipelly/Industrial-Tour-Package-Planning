Edu-Trip.

Plan With Us

It is a web based reservation system designed to make it easier for people/college to book industrial tour packages online. The most important part of this project is its database. Admin access the modules such as adding subjects, places, package details. System allows admin to view all the booking details & user details. User need to login and view various industrial tour & travel destinations with the package details & pricing for each person. If user wants to take multiple people on tour, then he/she needs to specify the number of people. Based on this, the system will automatically calculate the cost which is payable by user using online mode. After booking is successful, user receives an email notifying the booking details. User can view all their previous & current booking details..
